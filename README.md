## Developement
### setup env
```export mongo_uri=your_mongodb_connection_string```
### checkout 
```git clone https://github.com/shivshankar3578/limehome-assignment```
### install dependancies

```
cd limehome-assignment
npm install
```
### start local server
``` npm start```
